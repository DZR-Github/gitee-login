# gitee-login

> 使用Webpack5搭建的 gitee-login project



### Steps

1. 安装依赖  `npm  install`
2. 启动  `npm  run  serve`
3. 打开：localhost：3030



仓库地址：https://github.com/DZR-Github/gitee-login

项目部署地址：https://gitee-neon.vercel.app/
